# vectordbs
A python library that supports all vector databases specifically for LLM apps and frameworks

## List of Supported Vector databases 


| Task           | Supported |
|----------------|-----------|
| Chroma         |  :x:      |
| Pinecone       |  :x:      |
| Weaviate       |  :x:      |
| Qdrant         |  :x:      |
| Milvus         |  :x:      |
| Redis          |  :x:      |
| Opensearch     |  :x:      |
| Postgres       |  :x:      |
